Performs gene set overrepresentation analysis for cluster specific genes.

Output
For each clustering group and each cluster 3 tabels are produced: ora_table_all.csv with all genes, ora_table_up-regulated.csv with only up-regulated genes, and ora_table_all.csv with only down-regulated genes. The tables all contain the following columns:

  • Term: the ID of the respective pathway
  • Set size: the total number of genes in the pathway
  • Overlap ratio: the percentage of pathway related genes present in the DGE data
  • p-value: the p-value
  • FDR p-value: the FDR adjusted p-value
  • Odds ratio:
  • Combined score:
  • Features: the differentially expressed genes that overlap with the pathway

Additionally a barplot ora_barplot_fdr.svg is created showing the top pathways sorted by FDR.
