Performs gene set enrichment analysis for each cluster in each clustering group.

Output
For each clustering group and each cluster gsea_table.csv is produced containing the following columns:

  • source: the ID of the respective pathway
  • score: the enrichment score of the pathway
  • norm: the normalised enrichment score of the pathway
  • pval: the p-value for the enrichment

Additionally a barplot gsea_barplot_pval.svg is created showing the top pathways sorted by p-value.
