Aggregates cell-type annotated single-cell data into pseudo-bulk data for differential gene expression analysis.

Output
Each annotated sample is aggregated into a common pseudobulk.h5ad file. It contains the following additional observation variables:

  • sample: the sample name / type
  • replicate: the replicate or batch for the specified sample type
  • celltype: the cell type annotation of the aggregated sample
  • cellcount: the number of cells that were used to create the pseudo-bulk sample

References

  1. Heumos, L., Schaar, A.C., Lance, C. et al. Best practices for single-cell analysis across modalities. Nat Rev Genet (2023). https://doi.org/10.1038/s41576-023-00586-w
