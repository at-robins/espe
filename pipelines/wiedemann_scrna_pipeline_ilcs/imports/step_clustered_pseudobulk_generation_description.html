Aggregates cluster annotated single-cell data into pseudo-bulk data for differential gene expression analysis of the respective clusters.

Output
Each annotated sample is separated into its clusters and aggregated into a common cluster_pseudobulk.h5ad file. Additionally, each annotated sample is separated into data with each cluster removed respectively and aggregated into a common cluster_exclude_pseudobulk.h5ad file. Both files contain the following additional observation variables:

  • sample type: the sample type
  • replicate: the replicate for the specified sample type
  • clusteringgroup: the grouping that was used for clustering samples
  • cluster: the cluster that was used for / was excluded from pseudo-bluk generation
  • cellcount: the number of cells that were used to create the pseudo-bulk sample

References
