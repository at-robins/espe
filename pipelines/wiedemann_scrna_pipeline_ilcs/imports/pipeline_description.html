This pipeline processes pre-processed single cell RNA-Seq data specifically from pre-sorted group 1 ILCs. It is meant for internal usage and very specific.

Input specifications
Each sample needs to be provided as a separate folder containing a pre-processed filtered_feature_bc_matrix.h5ad file. Additionally, the following comma delimeted and UTF-8 encoded CSV files are required at the experiment's root directory:

  • sample_comparison.csv: This file defines which samples are compared to each other. An example can be downloaded here.
  • sample_clustering.csv: This file defines which samples to pool for clustering. Each row represents all samples of a common clustering pool. An example can be downloaded here.

System requirements
64 GB of RAM are recommended for pipeline execution.
Please also make sure you have sufficient storage space.
