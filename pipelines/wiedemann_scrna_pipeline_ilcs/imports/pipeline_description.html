This pipeline processes pre-processed single cell RNA-Seq data specifically from pre-sorted group 1 ILCs. It is meant for internal usage and very specific.

Input specifications
Each sample needs to be provided as a separate folder containing a pre-processed filtered_feature_bc_matrix.h5ad file. Additionally, a sample_comparison.csv (comma delimeted and UTF-8 encoded) at the directory root is needed that defines which samples are compared to each other. An example can be downloaded here.

System requirements
64 GB of RAM are recommended for pipeline execution.
Please also make sure you have sufficient storage space.
