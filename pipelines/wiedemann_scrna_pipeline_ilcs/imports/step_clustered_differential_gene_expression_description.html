Calculates differentially expressed genes of specific clusters and of clusteres between sample types.

Output
For each clustering group and each cluster a respective subfolder is generated at cluster_comparison/{clustering_group}/{cluster}. Similarly, for each sample type / condition comparison specified in the sample_comparison.csv and each cluster a respective subfolder is generated at sample_comparison/{sample_type_1}__vs__{sample_type_2}/{cluster}. Each subfolder contains the following files:

  • differential_gene_expression.csv: a CSV containing differentially expressed genes including log2 fold change, p value and FDR
  • info.csv: a CSV file containing information related to the differential gene expression analysis
  • mds_plot_plot.svg: a multi-dimensional scaling plot of the pseudobulk data (similar samples are expected to be located in close proximity)
  • bcv_plot.svg: a biological coefficient of variation plot that shows gene expression variability
  • smear_plot.svg: a smear plot that shows change in expression against abundance
