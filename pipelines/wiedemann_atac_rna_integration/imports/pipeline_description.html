This pipeline integrates ATAC and RNA-Seq data.

Input specifications
The pipeline expects sub-folders, that each contain two following comma delimeted and UTF-8 encoded differential gene expression / differential accessibility CSV files:

  • atac_da.csv: The respective ATAC-Seq DA information. Must contain the following columns: Peak ID, Gene symbol, Annotation, log2FC, FDR
  • rna_dge.csv: The respective RNA-Seq DGE information. Must contain the following columns: Gene symbol, log2FC, FDR

System requirements
No specific requirements.
