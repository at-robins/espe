This pipeline processes paired end ATAC data.

Input specifications
The input is expected as gzipped FASTQ files (.fq.gz / .fastq.gz). Pairs of read files must always reside in the same sub-folder and end with 1 or 2 respectively. Additionally, the following comma delimeted and UTF-8 encoded CSV files are required at the experiment's root directory:

  • sample_information.csv: This file defines sample naming, replicate detection and batch correction. An example can be downloaded here.
  • sample_comparison.csv: This file defines which samples are compared to each other. An example can be downloaded here.

System requirements
64 GB of RAM are recommended for pipeline execution.
