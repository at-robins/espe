Visualises peak data.

Output The following files are created:

  1. clustermap.svg: A clustermap comparing the normalised counts of all samples.
  2. genomic_regions_barplot_*.svg: A barplot for each sample condition comparison, that visualises changes per genomic feature type (exons, introns, ...).
  3. volcanoplot_*.svg: A volcano plot for each sample condition comparison that correlates FDR and overall change of identified peaks.
