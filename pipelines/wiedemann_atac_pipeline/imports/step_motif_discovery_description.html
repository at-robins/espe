Finds enriched motifs known and de novo in differentially accessible peak regions.

Output
For every differantial accesibility comparison motif detection using HOMER is run. For details on the output see HOMER's documentation section findMotifsGenome.pl Output. Additionally, the dicovered motifs are mapped back on the respective peaks. This information can be found in the according motif_annotated_peaks.tsv files (UTF-8 encoded; tab delimited). For details on the output format see HOMER's documentation section Finding Instance of Specific Motifs.
