Performs pathway analysis for each sample comparison.

Output
For each sample comparison and used pathway databasea pathway_analysis_{pathway database}_table.csvs is produced. It contains the following columns:

  • pathway: the ID of the respective pathway
  • score: the enrichment score of the pathway
  • pvalue: the p-value for the enrichment

Additionally, an extended table pathway_analysis_{pathway database}_genes.csv is produced that contains differential peak accesibility information for every signifanctly altered pathway. It contains the following columns:

  • source: the ID of the respective pathway
  • target: the symbol of the associated gene
  • weight: the influence strength of the associated gene on the pathway
  • log2FoldChange: the log-2-fold change in accessibility
  • pvalue: the p-value for the enrichment
  • padj: the adjusted p-value for the enrichment
  • PeakID: the ID of the respcetive peak

Finally, a barplot pathway_analysis_{pathway database}_barplot.svg is created showing significant (p-value ≤ 0.05) pathway alterations.
