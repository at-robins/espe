Determines differentailly accessible regions between conditions based on the input sample_comparison.csv.

Output The following files are created:

  1. pca.svg: A dotplot of the principle component analysis of all different samples.
  2. differential_accessibility_*.csv: A UTF-8 encoded CSV for each condition comparison containing differentially accessible peak region analysis results.
  3. annotated_differential_accessibility_*.csv: The respective differential_accessibility_*.csv files with additional peak annotation information as generated during the peak annotation step.
  4. ma_*.csv: The MA plot of the respective sample comparison.

References

  1. Love, M. I., Huber, W., & Anders, S. (2014). Moderated estimation of fold change and dispersion for RNA-seq data with DESeq2. Genome biology, 15(12), 550. https://doi.org/10.1186/s13059-014-0550-8
