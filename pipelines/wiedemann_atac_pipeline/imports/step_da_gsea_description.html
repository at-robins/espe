Performs gene set enrichment analysis for each sample comparison.

Output
For each sample comparison and used pathway database a gsea_{pathway database}_table.csv is produced containing the following columns:

  • source: the ID of the respective pathway
  • score: the enrichment score of the pathway
  • norm: the normalised enrichment score of the pathway
  • pval: the p-value for the enrichment

Additionally, an extended table gsea_{pathway database}_table.csv is produced that contains differential peak accesibility information for every signifanctly altered pathway. It contains the following columns:

  • geneset: the ID of the respective pathway
  • genesymbol: the symble of the associated gene
  • logFC: the log-2-fold change in accessibility
  • PValue: the p-value for the enrichment
  • FDR: the FDR for the enrichment
  • PeakID: the ID of the respcetive peak

Finally, a barplot gsea_{pathway database}_barplot.svg is created showing the top pathways sorted by p-value.
