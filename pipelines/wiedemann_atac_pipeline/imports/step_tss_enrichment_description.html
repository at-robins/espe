Plots read density around transcription start sites (TSS).

Output The following files are created for each sample and each nucleosomal fraction (nucleosome-free, mononucleosomal, ...):

  1. *.bw: A bigWig file representation of the input bam file.
  2. *.mat.gz: A zipped matrix file containg count distribution information.
  3. *_heatmap.svg: A heatmap of the read distribution around the TSS.
  4. *_read_density.svg: A density plot of the read distribution around the TSS.
