Clusters separated samples at a default resolution.

Output
For each clustering group the following files are generated:

  • clustered.h5ad: the concatenated sample data including clustering information
  • default_umap.svg: a default UMAP visualisation of the clustering, sample types, replicates and quality control metrics for QC purposes
  • default_marker_genes.svg: a dotplot showing marker genes for the respective clusteres depicted in the QC UMAP
  • pca_variance.svg: PCA variance plot of the data
  • sample_{sample type}.svg: the umap representation with the specific sample type highlighted
  • replicate_{replicate name}.svg: the umap representation with the specific replicate highlighted
