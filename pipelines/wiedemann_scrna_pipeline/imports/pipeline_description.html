This pipeline processes single cell RNA-Seq data. For data obtained with fixed cell protocols it is recommended to skip the ambient RNA removal step by using the according step variable below.

Input specifications
Each sample needs to be provided as a separate folder containing the filtered and raw H5 count matrices that are output by CellRanger . The files need to contain the characters filtered_feature_bc_matrix.h5 and raw_feature_bc_matrix.h5 respectively. H5AD files are also supported. raw_feature_bc_matrix.h5 is only needed if ambient RNA removal is required. If no raw data is available or the data is expected to contain very low amounts of ambient RNA, please skip the ambient RNA removal step (see the varibles section) otherwise errors are to be expected.
Additionally, a sample_information.csv (comma delimeted and UTF-8 encoded) at the directory root is needed that is used for sample naming, replicate detection and batch correction. An example can be downloaded here. The following columns must be specified:

  • sample directory: the name of the directory the sample files reside in.
  • sample type: the type of sample used to form comparable conditions, e.g. treatment and control

System requirements
64 GB of RAM are recommended for pipeline execution.
Please also make sure you have sufficient storage space.
