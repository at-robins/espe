Merges the differential gene expression information by traversing the cluster resolution tree. Enrichment of a differentially expressed gene for a specific cluster is analysed by checking during how many splits it is relevant and how early these splits happen.

Output
A raw ({cluster_id}_merged_dge_raw.csv) and filtered DEG enrichment list ({cluster_id}_merged_dge_filtered.csv) per cluster and cluster split event is generated. The filtered version does not contain genes that were never differentially expressed for the specific cluster. The following columns are present in each file per gene:

  1. raw enrichment: the sum of log2 fold change of the gene over all splitting events relevant for this cluster
  2. signigicance: the latest split event when this gene was differentially expressed (-1 if never differentially expressed)
  3. weighted enrichment: the sum of cluster size and split event weighted log2 fold changes of a gene over all splitting events relevant for this cluster
