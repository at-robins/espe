Scores cells based on marker genes defined in the optional input file celltypes.csv. The file must contain the following 3 columns (for more information see decoupler ULM net parameter):

  • source: the respective cell type / state name
  • target: a gene used to score the cell type or state defined in source
  • weight: the weight of the gene defined in target (between 1.0 [strong positve marker] and -1.0 [strong negative marker])

Output
Multiple cell_types_{chunk_index}.svg files are created containing UMAPS with the respective ULM scores for cell types / states defined in source.
