Estimates likely cluster splits and generates a respective cluster tree.

Output
For each clustering group a genealogy_cluster_resolution_data.json file is generated containg information of all estimated cluster splits and their resolutions. Additionally, a stability_graph_cluster_resolution_data.svg a plot of the inferred cluster stability is generated for cutoff visualisation.
