Performs an initial quality control of sequenced reads and removes low quality cells.

Output
For each sample the preprocessed filtered_feature_bc_matrix.h5ad file is generated. Metrics are summed up in the metrics.csv file. Additionally the following files are generated per sample:

  • ddqc_box_plot_genes.svg: Boxplot of the number of genes per QC cluster. The conventionally used thershold of 200 genes is marked with a red line.
  • ddqc_box_plot_mitochondrial_reads.svg: Boxplot of the percentage of mitochondiral reads per QC cluster. The conventionally used thershold of 10 % is marked with a red line.
  • ddqc_line_plot_absolute.svg: Line plot of the number of cells per QC cluster that would be filtered out when setting the MAD threashold to the respective x-axis value. The threshold is marked with a dotted line.
  • ddqc_line_plot_relative.svg: Line plot of the percentage of cells per QC cluster that would be filtered out when setting the MAD threashold to the respective x-axis value. The threshold is marked with a dotted line.
  • ddqc_table.csv: Table containg the QC results. For further information see the ddqc documentation.
